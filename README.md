# Yes-Chef---Tentworks-Interactive-Dev-T-est

The game's architecture uses ScriptableObjects for ingredients and preparation stations ( stove / cutting board ) so that a designer can easily extend the game as much as they want without writing any code.

Open Scenes -> Yes Chef and play

Controls
  Move : WASD / Arrow Keys
  Interact : E (near a station)
  Pause : Escape
  
Walk to the Refrigerator the ingredient menu opens automatically.
Prepare ingredients at the Table (chop) or Stove (cook).
Deliver to a Customer Window and press E to serve.
Use the  Trash to discard wrong ingredients.
